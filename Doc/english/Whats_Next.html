Features and Future

BeBuilder V0.5 is still a pre release. We are still interested in any comments, even the bad ones. :-)
  • All the BControl included in the BeOS Documentation

    As we said since BeBuilder V0.1 we are working hard to add all the components included in BeOS, into BeBuilder. It's just a question of time! 8D

  • Project Options

    Since V0.5 you can save and load projects.

  • GCC Integration

    The Tools menu has been upgraded but is still experimental you can make, run and edit your files from BeBuilder, sorry for the Pe and Eddie lovers only the BeIDE is supported for the moment.

  • Dynamic Code Generation

    As you might certainly have noticed there is no moreGenerate option in the menu. Since V0.5 the code generation is dynamic as soon as you drop a component or modify a property the source code is modified and your modifications should not be lost.
    I said should to take no risks but BeBuilder is developed with BeBuilder...
    RULE OF THUMB : NEVER EVER MODIFY A CODE LINE WRITTEN BY BEBUILDER.

  • Bugs

    You may find bugs... :) don't hesitate to E-mail me concerning those bugs and if possible include a description or a simple project that crashes BeBuilder

  • Future Editor

    We hope to create a simple editor which source code will be provided to support some cool features of BeBuilder.

  • Openings

    For the moment the code generation is only C++ but we could imagine to add other languages.... make your choice!
