Introduction

BeBuilder is a GUI builder and C++ Generator for BeOS.
The code generated in BeBuilder is without any overhead, nor library nor statically linked resources.
The code is PURE, only C++ and BeOS API.
BeBuilder generates code as you would write it!

If you are not statisfied by BeBuilder you can do what you want with the source code generated... no overhead, no library, no resource... just simple Object Oriented C++ code.
In those times of "OPEN stuff (tm)" BeBuilder is open minded! You will never be stuck with it!
