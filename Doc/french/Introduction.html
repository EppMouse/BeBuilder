Introduction

BeBuilder est un créateur dèinterfaces et un générateur de code C++ pour BeOS.
Le code généré par BeBuilder est sans ajout de code, ni librairie ajoutée à l'édition de liens, ni ressource spéciale.
Le code est considéré comme PUR, seulement du C++ et des API natives BeOS.
BeBuilder génère le code comme vous pourriez l'écrire!

Si vous n'êtes pas satisfait de BeBuilder, vous pouvez récupérer très facilement le code généré... Pas d'ajout de code, pas de librairie, pas de resource... du simple code C++ orienté objet.
Dans ces temps de "OPEN qqchose (tm)" BeBuilder a l'esprist ouvert! Vous ne serez jamais pogné avec (tm)!!
